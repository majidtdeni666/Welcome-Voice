☣️ Installation
- pkg update -y
- pkg upgrade -y
- pkg install mpv -y
- pkg install git -y
- git clone https://github.com/majidtdeni666/Welcome-Voice.git
- cd Welcome-Voice
- chmod +x *
- sh install.sh

Now Type exit & restart Termux
